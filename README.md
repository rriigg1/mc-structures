# Minecraft structure vierwer
A web application that can display minecraft structure files. The whole application is client sided and no data is uploaded to the server.
The textures are sourced from a resource pack available for download online.

Try it now: [Github-Pages](https://rriigg1.github.io/mc-structures/)

### Current features:
- Upload your own structure file
- Currently only full blocks are displayed and all sides are using the same texture

### Planned:
- Support for blocks with different textures per side (logs, furnace, ...)
- Support for non full blocks
- Selection of resource pack
