# Minecraft structure vierwer
A web application that can display minecraft structure files. The whole application is client sided and no data is uploaded to the server.
The textures are sourced from a resource pack available online.

Try it now: [Github-Pages](https://rriigg1.github.io/mc-structures/)

---

${\color{red}\huge{\textsf{Disclaimer: The code was written with the assistance of LLMs!}}}$


### Current features:
- Upload your own structure file
- Most blocks should work
    - Full blocks
    - Block variants
    - Mulitpart blocks
    - Tinted textures
- Generation of basic outline for a house

### Known issues:
- Half-transparency not working correctly (stained glass)
